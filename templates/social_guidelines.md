# Social Media Guidelines for Scientific Publishing

## Character Limits & Platform Guidelines

### Short-form (Hook-focused)
- **Twitter/X:** 280 characters - punchy, hashtags, engaging hook
- **Bluesky:** 300 characters - similar to Twitter, slightly more formal
- **Threads:** 500 characters - conversational, can be more casual
- **Mastodon:** 500 characters - technical audience, science-friendly

### Long-form (Educational)
- **Reddit:** Title (300 chars) + Body (40,000 chars) - detailed explanation, ELI5 style for r/science or technical for r/programming
- **Facebook:** 63,206 characters (keep < 500 for engagement) - accessible to general audience
- **LinkedIn:** 3000 characters (keep < 1000) - professional tone, emphasize methodology
- **Instagram:** 2200 characters - visual-first, caption supports the image, use line breaks

## The Unicode Constraint (STRICT)

**CRITICAL:** Do NOT use LaTeX syntax (e.g., `$\frac{a}{b}$`) in social posts.
Standard social platforms DO NOT render LaTeX. It appears as raw code.

Use Unicode approximations instead:

| LaTeX | Unicode |
|-------|---------|
| `\alpha` | α |
| `\beta` | β |
| `\gamma` | γ |
| `\delta` | δ |
| `\pi` | π |
| `\int` | ∫ |
| `\sum` | ∑ |
| `\rightarrow` | → |
| `\approx` | ≈ |
| `\infty` | ∞ |
| `x^2` | x² |
| `x^3` | x³ |
| `\sqrt` | √ |
| `\vec{v}` | **v** (bold) |

**Rule:** If a symbol cannot be converted to Unicode, rewrite the sentence in plain English.
