- **LaTeX Engine**: pdflatex

- **Python Libraries**: numpy, scipy, matplotlib
- **Packages**: amsmath, amssymb, graphicx, booktabs, siunitx, geometry, hyperref, float

## Compilation Process

Each template requires a 3-pass compilation:

```bash
pdflatex template.tex      # Pass 1: Extract Python code
pythontex template.tex     # Pass 2: Execute Python, generate plots
pdflatex template.tex      # Pass 3: Include results and plots
```

Automated compilation script: `/home/user/latex-templates/compile_new_100_templates.sh`
