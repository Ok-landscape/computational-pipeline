Title: Exploring the Gamma Function with Python - Extending Factorials to All Numbers

Body:

**What is the Gamma Function?**

The Gamma function Γ(z) is one of mathematics' most important special functions. It answers the question: what is 0.5 factorial?

**The Definition**

Γ(z) = ∫₀^∞ t^(z-1) e^(-t) dt

This integral converges for any z with positive real part.

**Key Properties**

1. **Factorial Connection**: Γ(n+1) = n! for positive integers
2. **Recurrence**: Γ(z+1) = z·Γ(z)
3. **Special Value**: Γ(1/2) = √π ≈ 1.7725
4. **Reflection Formula**: Γ(z)Γ(1-z) = π/sin(πz)
5. **Duplication Formula**: Γ(z)Γ(z+1/2) = √π·Γ(2z)/2^(2z-1)

**What I Learned**

- The function has poles (singularities) at 0, -1, -2, -3, ...
- Between poles, it alternates between positive and negative
- Stirling's approximation works well for large values
- The Beta function B(a,b) = Γ(a)Γ(b)/Γ(a+b)

**Why It Matters**

The Gamma function appears everywhere:
- Gamma probability distribution
- Quantum mechanics wave functions
- String theory amplitudes
- Generalizing binomial coefficients
