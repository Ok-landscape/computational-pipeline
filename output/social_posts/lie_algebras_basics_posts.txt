I built an interactive notebook exploring **Lie algebras** - the mathematical structures that describe continuous symmetries in physics. Here's what I learned:

**What's a Lie algebra?**

Think of it as a vector space with a special operation called the "bracket" [A,B] = AB - BA (for matrices, this is the commutator). This bracket must satisfy:

1. Bilinearity (linear in both arguments)
2. Antisymmetry: [X,Y] = -[Y,X]
3. Jacobi identity: [X,[Y,Z]] + [Y,[Z,X]] + [Z,[X,Y]] = 0

**The Cool Part: so(3)**

The algebra so(3) consists of 3x3 skew-symmetric matrices. Its generators Lx, Ly, Lz satisfy:
- [Lx, Ly] = Lz
- [Ly, Lz] = Lx
- [Lz, Lx] = Ly

These are infinitesimal rotations! Using the exponential map exp(theta * L), you get actual rotation matrices in SO(3).

**Code highlights:**

```python
def lie_bracket(A, B):
    return A @ B - B @ A

def verify_jacobi_identity(X, Y, Z):
    term1 = lie_bracket(X, lie_bracket(Y, Z))
    term2 = lie_bracket(Y, lie_bracket(Z, X))
    term3 = lie_bracket(Z, lie_bracket(X, Y))
    return np.allclose(term1 + term2 + term3, 0)
```

**Why it matters:**

- so(3): Angular momentum in mechanics
- su(2): Quantum spin (Pauli matrices!)
- su(3): Quantum chromodynamics (the strong force)
