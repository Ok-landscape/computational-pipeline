**Title:** I built a Python calculator for the Riemann curvature tensor - the math behind Einstein's General Relativity

**Body:**

The Riemann curvature tensor is arguably the most important object in differential geometry. It measures the intrinsic curvature of a space - properties that can be detected without leaving the surface.

**The key intuition:** If you parallel transport a vector around a closed loop in flat space, it returns unchanged. In curved space, it rotates! The Riemann tensor quantifies exactly how much.

**What my code does:**

1. Takes any metric tensor g_mn as input
2. Computes Christoffel symbols using numerical differentiation
3. Calculates the full Riemann tensor R^p_smn
4. Derives contractions: Ricci tensor, Ricci scalar, Kretschmann scalar

**Cool results:**

- **Sphere:** Gaussian curvature K = 1/R² everywhere (constant positive curvature)
- **Torus:** K > 0 on outer edge, K < 0 on inner edge, with integral K dA = 0 (Gauss-Bonnet theorem!)
- **Schwarzschild black hole:** The Kretschmann scalar K = 48M²/r⁶ diverges at r=0, proving it's a true singularity (not a coordinate artifact like the event horizon)

**Symmetry verification:**

The code numerically confirms all Riemann tensor symmetries:
- R_psmn = -R_psnm (antisymmetric in last two indices)
- R_psmn = -R_spmn (antisymmetric in first two indices)
- R_psmn = R_mnps (pair symmetry)
- R_psmn + R_pmns + R_pnsm = 0 (first Bianchi identity)

The visualization shows parallel transport on a sphere - a vector carried around a triangular path (pole to equator to pole) rotates 90 degrees, directly demonstrating curvature.
