**Title:** Visualizing Taylor Series Convergence in Python - How Your Calculator Actually Computes sin(x)

**Body:**

Ever wondered how calculators evaluate transcendental functions like eˣ or sin(x)? They use Taylor series—polynomial approximations that converge to the exact function value.

**What is a Taylor Series?**

For any smooth function f(x) at point a:

f(x) = f(a) + f'(a)(x-a) + f''(a)/2! · (x-a)² + f'''(a)/3! · (x-a)³ + ...

Or more compactly: f(x) = Σ f⁽ⁿ⁾(a)/n! · (x-a)ⁿ for n = 0 to ∞

**Key Examples:**

• eˣ = 1 + x + x²/2! + x³/3! + ...
• sin(x) = x - x³/3! + x⁵/5! - ...
• cos(x) = 1 - x²/2! + x⁴/4! - ...

**What I Learned:**

1. Error decreases exponentially with more terms (for functions with infinite radius of convergence)
2. ln(x) expanded around a=1 only converges for 0 < x ≤ 2
3. Factorials in denominators make convergence surprisingly fast

**Error Analysis:**

At x=1, approximating e with just 11 terms gives error < 10⁻¹⁰. The factorial growth in denominators dominates the polynomial growth in numerators.

**Numerical Methods in Practice: Taylor Series Approximations**

Taylor series are foundational to numerical computing, enabling efficient evaluation of transcendental functions in everything from scientific calculators to machine learning libraries.

In this computational notebook, I implemented and analyzed Taylor series approximations for:
• Exponential function (eˣ)
• Trigonometric functions (sin, cos)
• Natural logarithm (ln)

**Key Technical Findings:**

The error analysis demonstrates exponential convergence—approximating e with 11 terms achieves error < 10⁻¹⁰. This is because factorial growth in denominators (n!) dominates polynomial growth in numerators (xⁿ).

The logarithm case illustrates an important constraint: Taylor series have finite radii of convergence. For ln(x) expanded around a=1, the series only converges for 0 < x ≤ 2.
