**Title:** Exploring the Laplace Transform with Python: From Theory to Visualization

**Body:**

The Laplace transform is one of the most powerful tools in engineering mathematics. It converts functions of time f(t) into functions of complex frequency F(s):

L{f(t)} = F(s) = ∫₀^∞ f(t)·e^(-st) dt

**Why is this useful?**

It transforms differential equations into algebraic equations. Instead of solving:

y'' + 2ζωₙy' + ωₙ²y = ωₙ²u(t)

You can work with the transfer function:

H(s) = ωₙ² / (s² + 2ζωₙs + ωₙ²)

**What I explored:**

1. **Numerical Laplace transforms** - Computing transforms via numerical integration
2. **Transform pairs** - Visualizing exponentials, sine waves, and damped oscillations
3. **Step responses** - How damping ratio ζ affects system behavior (underdamped oscillates, overdamped is sluggish)
4. **Pole-zero analysis** - Why poles in the left half-plane mean stability
5. **Bode plots** - Frequency response in magnitude (dB) and phase
6. **Convolution theorem** - Time-domain convolution = frequency-domain multiplication

**Key insight:** A system is stable if and only if ALL poles have negative real parts (left side of s-plane).

Built with NumPy, SciPy, and Matplotlib. The notebook demonstrates how these abstract mathematical concepts connect to real engineering applications in control systems, signal processing, and circuit analysis.

Exploring the Laplace Transform: A Computational Approach

The Laplace transform remains a cornerstone of engineering analysis, converting time-domain problems into algebraic ones that are more tractable.

In this computational notebook, I implemented:

• Numerical Laplace transform computation using SciPy's integration tools
• Visualization of common transform pairs (exponential, sinusoidal, damped oscillations)
• Second-order system analysis with varying damping ratios
• Pole-zero plots demonstrating stability criteria

• Convolution theorem verification

Key engineering insight: System stability is determined entirely by pole locations. Poles in the left half of the s-plane (negative real parts) guarantee stability—a principle fundamental to control system design.

The transfer function H(s) = ωₙ²/(s² + 2ζωₙs + ωₙ²) elegantly captures how natural frequency and damping ratio shape system response.
